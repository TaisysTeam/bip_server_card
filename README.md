# BIP Server for JavaCard Connect

An object oriented PHP wrapper for the JavaCard API

## Requirements

- PHP >= 7.4
- Nginx >= 1.14.1
- Mysql >= 8.0
- Redis >= 6.0.8

## Usage

setup your Nginx on you Server and put this program in you root directory.
